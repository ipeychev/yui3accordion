What is Accordion widget?

Accordion widget

Accordion widget allows grouping items, containing arbitrary data inside. These items can be expanded, collapsed, closed, reordered by using drag and drop, etc.
The height of the content inside each item can be set in three different ways:

  1. auto - this means the height will be as is being calculated from browser
  2. fixed - the content height will be exactly X pixels
  3. stretch - the height will be calculated, based on the height of the other items in the Accordion

  3. Dynamically add or remove items

  4. Overwrite the default properties
